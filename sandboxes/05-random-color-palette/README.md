# 05-random-color-palette

Generates cohesive five-color palettes at random using a few harmonic strategies (spread hues, monochrome ramp, analogous plus complement). Click a swatch to copy hex; Space or **New palette** reshuffles.

## Run

```bash
# from the repo root:
bun dev 05-random-color-palette

# or from this folder:
bun run dev
```

## Files

- `index.html` - page shell
- `src/main.ts` - palette logic and clipboard
- `src/styles.css` - layout
- `vite.config.ts` - Vite settings

## Add a dependency

```bash
bun add --filter @sandbox/05-random-color-palette <pkg>
```
